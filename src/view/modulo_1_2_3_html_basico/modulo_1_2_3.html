Estrutura

          
            <html lang="pt-br">
              <head>
                <meta charset="UTF-8" />
                <meta name="viewport" content="width=device-width, initial-scale=1.0" />
                <title>Exemplo de código HTML5</title>
              </head>
              <body>
                <h1>Titulo</h1>
                <p>Isso é um parágrafo.</p>
                <!-- comentario -->
              </body>
            </html>

Essa é a estrutura basica do HTML5.

  • Na primeira linha temos a tag <html> indicando que o documento é HTML seguido pelo parametro lang="pt-br" que indica que a página será escrita em português do Brasil (perceba que lang indica a linguagem em relação ao conteúdo e não em relação ao código).
  • Em seguida temos a abertuda da tag <head>, dentro dessa tag colocamos as informações meta, title, links, stylesheets e etc.
  • No exemplo temos <meta charset="UTF-8"/>. A tag <meta> é uma tag que abre e fecha em si mesma e fornece metadados sobre o documento (informações que descrevem outros dados e não são exibidos diretamente na página, mas influenciam como a página é apresentada e interpretada). Nesse no caso, indica que o conteúdo da pagina será escrito usando os padrões de digitação UTF-8.
  • Em sequência temos <meta name="viewport" content="width=device-width, initial-scale=1.0"/> que é usada no HTML para controlar a forma como uma página web é exibida em diferentes dispositivos,
    • O atributo name com o valor "viewport" especifica que esta meta tag está definindo as propriedades da "janela de visualização" (viewport) da página.
    • O atributo content contém uma lista de propriedades e valores separados por vírgulas que controlam a escala e a largura da viewport
      • width=device-width define a largura da viewport para ser igual à largura da tela do dispositivo. Isso significa que a largura da página será ajustada automaticamente para se adequar à largura do dispositivo em que está sendo visualizada. É especialmente útil para garantir que a página seja corretamente dimensionada em dispositivos móveis.
      • Enquanto initial-scale=1.0 define o nível inicial de zoom da página quando carregada pela primeira vez. Um valor de 1.0 significa que a página será exibida com um zoom de 100%, ou seja, sem ampliação ou redução.
  • Por fim temos a tag <title>Exemplo de código HTML5</title>, que dita qual o titulo da página, esse tituço ficará escrito na aba do navegador.
  • Após fecharmos a tag <head>, é aberto a tag <body>, nesta ficará todo o conteúdo da página. No exemplo temos um titulo, um parágrafo e um comentário, que serão explicados a seguir:

Isso é um Título, é feito usando a tag "<h1>texto do título</h1>" pra fazer o título da página

Paragráfos, linhas, quebras e comentários

Isso é um paragráfo, é feito usando a tag <p> texto do parágrafo </p>

Um paragrafo pode ser quebrado usando a tag <br> e linhas horizontais são feitas usando a tag <hr>

Comentarios são Textos no código que não aparecem na página (são feito para desenvolvedores lerem), em html, comentarios são feito assim: <!-- seu comentario -->

Em html podemos fazer algumas formatações:

Negrito: <b>texto</b>, tambem podemos usar a tag <strong>texto</strong>, ambas tem o mesmo resultado visual porem usando strong acrescenta um pouco mais de semantica (dizendo que o texto tem mais força) para o navegador e mecanismos de busca.

Italico: <i>texto</i> e tambem podemos usar a tag <em>texto</em>, ambas tem o mesmo resultado visual porem usando em acrescenta um pouco mais de semantica (dizendo que o texto tem enfase) para o navegador e mecanismos de busca.

Marcação: <mark>texto</mark>, usado para deixar o texto marcado

nota de rodapé:: <small>texto</small> Usado para deixar o texto um pouquinho menor, pode ser usado em notas de rodapé

Subscrito: H<sub>2</sub>O Usado para deixar o texto subscrito (H2O)

Superscrito: 2<sup>2</sup>=4 Usado para deixar o texto subscrito (22=4)

Links e Imagens

Isso é um Link: Links são como meios de navegação entre paginas, cada link possui um endereço pra onde quer ir, e um texto. assim:
<a href="./endereco">texto do link</a>

Esse link abre o google em uma nova aba: pra que isso seja feito devemos colocar o parametro target="_blank" assim:
<a href="https://www.google.com" target="_blank">texto do link</a>

Tel: +55 67 9 9200-1234: Há a possibilidade de fazermos um link para um numero de telefone, pra isso precisamos colocar tel: antes do numero no endereço do link ficando assim:
<a href="tel:+5567992001234">+55 67 9 9200-1234:</a>

Whatsapp: +55 67 9 9200-1234: O whatapp tem a possibilidade de fazer links que direcionam para uma conversa, pra isso precisamos usar o padrão https://wa.me/numero-do-whatsapp, o codigo fica assim:
<a https://wa.me/5567992001234" target="_blank">+55 67 9 9200-1234:</a>

Instagram: @matheus.rcg O instagram também tem a possibilidade de fazer links que direcionam direto para o aplicativo (caso o tenha instalado no dispositivo), pra isso usamos um link personalizado que fica assim:
<a href="intagram://user?username=nome-do-usuario">texto do link</a>

Email: meu.email@email.com Pra fazermos links para email, acrescentamos mailto: ao endereço do link, isso fará o usuario ser direcionado ao app padrão de abrir emails, o código fica assim:
<a href="mailto:endereco_do_email@email.com">texto do email</a>

Imagens são inseridas através da tag img, nela temos o parametro src que será passado o endereço da imagem (seja da internet ou dos próprios arquivos), outro parametro a ser passado é o alt que é um texto alternativo normalmente descrevendo a imagem, esse texto aparecerá no caso do endereço não ser encontrado. podemos tambem passar width e height para redimencionar a imagem (porém é mais indicado que isso seja feito usando CSS)
<img src="./Images/html-logo.png" alt="Logo do HTML"width="75px">

Dentro dessa tag colocamos a imagem com a tag img e a legenda é feita com <figcaption>legenda</figcaption>, o resultado fica assim: 

          
            <figure>
              <img src="./Images/html-logo.png" alt="Logo do HTML" width="100px">
              <figcaption>
                Texto da legenda
              </figcaption>
            </figure>

É possivel também inserir uma imagem com link, pra isso precisamos colocar a tag <img> no conteúdo da tag <a>, o codígo ficaria assim: 

          
            <a href="./endereco/do-link">
              <img src="./endereco/da/imagem.extencao" alt="texto alternativo">
            </a>

Listas

Existem 2 Tipos de listas, as listas ordenadas e as não ordenadas:

  1. Listas ordenadas são feitas usando a tag <ol>itens</ol>
  2. Listas não ordenadas são feitas usando a tag <ul>itens</ul>
  3. Para inserir itens em ambas as listas precisamos usar a tag <li>conteúdo</li>
  1. Para inverter a lista podemos usar o parametro reversed
  2. o codigo fica assim: <ol reversed>itens</ol>
  1. podemos tambem dizer onde a lista iniciará usando o parametro start
  2. o codigo fica assim: <ol start="15">itens</ol>
  1. Há a possibilidade também de dizer o tipo de lista que queremos usando type
  2. o codigo fica assim: <ol type="a">itens</ol>
  3. Os tipos de listas que podemos ter são:
    1. Alfabeto maiusculo usando Type="A"
    1. Alfabeto minusculo usando Type="a"
    1. Numeros romano maiusculo usando Type="I"
    1. Numeros romano minusculo usando Type="i"
    1. E o que ja conhecemos que é o numeral
  4. Pra fazer hierarquia de itens precisamos colocar uma lista dentro de outra, o código fica assim: 
          
            <ul>
              <li>
                item
                <ul>
                  <li>
                    item dentro do item
                  </li>
                </ul>
              </li>
            </ul>

Resultado:

  • item
    • item dentro do item

DIV's e ID's

A tag <div> cria um contêiner genérico que é utilizado para agrupar outros elementos HTML e aplicar estilos ou scripts a eles de forma conjunta. "Div" é uma abreviação de "division" (divisão)

Id's é um parametro adicionado a elementos html para que estes elementos possam ser posteriormente referenciados

Aqui temos um exemplo de uma <div> junto com um id que contem uma lista de links que referenciam uma <div> pelo seu id 

          
            <div id="indice">
              <ul>
                <li>
                  <a href="#nome_do_id">texto de referencia</a>
                </li>
                <li>
                  <a href="#div_exemplo">Div Exemplo</a>
                </li>
              </ul>
            </div>
            <div id="nome_do_id">
              <h3>um titulo qualquer</h3>
            </div>
            <div id="div_exemplo">
              <h3>Div Exemplo</h3>
            </div>

esse é o resultado: (foi acrescentado texto as divs para melhor compreenção)

  • texto de referencia
  • Div Exemplo
