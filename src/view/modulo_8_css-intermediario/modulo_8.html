Tipos de Display

A propriedade display define como um elemento é exibido na página. Existem vários valores possíveis para essa propriedade, mas os mais comuns são block, inline, inline-block e none.

Display Block

Um elemento com display: block ocupa toda a largura disponível e começa em uma nova linha. Exemplos de elementos block são <div>, <p> e <h1>. 

          
            <div style="display: block; background-color: lightblue;">
              Elemento Block
            </div>
Elemento Block

Display Inline

Um elemento com display: inline ocupa apenas a largura necessária e não começa em uma nova linha. Exemplos de elementos inline são <span> e <a>. 

          
            <span style="display: inline; background-color: lightblue;">
              Elemento Inline
            </span>
Elemento Inline

Display Inline-Block

Um elemento com display: inline-block é semelhante ao inline, mas permite definir largura e altura. Ele não começa em uma nova linha. 

          
            <div style="display: inline-block; width: 100px; height: 50px; background-color: lightblue;">
              Elemento Inline-Block
            </div>
Elemento Inline-Block

Display None

Um elemento com display: none não é exibido na página e não ocupa espaço. 

          
            <div style="display: none;">Elemento None</div>
Elemento None

O elemento acima não é exibido na página.

Tipos de Position

A propriedade position define como um elemento é posicionado na página. Existem vários valores possíveis para essa propriedade, mas os mais comuns são static, relative, absolute, fixed e sticky.

Position Static

Um elemento com position: static é posicionado de acordo com o fluxo normal da página. Este é o valor padrão para todos os elementos. 

          
            <div style="position: static; background-color: red; width: 100px; height: 100px;"></div>
            <div style="position: static; background-color: green; width: 100px; height: 100px;"></div>
            <div style="position: static; background-color: blue; width: 100px; height: 100px;"></div>

Position Relative

Um elemento com position: relative é posicionado de acordo com o fluxo normal da página, mas pode ser deslocado em relação à sua posição original usando as propriedades top, right, bottom e left. 

          
            .div-azul {
              position: relative;
              top: 20px;
              left: 20px;
            }

Position Absolute

Um elemento com position: absolute é posicionado em relação ao seu ancestral mais próximo que não tenha position: static. Se não houver tal ancestral, ele será posicionado em relação ao elemento html. 

          
            .div-azul {
              position: absolute;
              top: 50px;
              right: 0;
            }

Position Fixed

Um elemento com position: fixed é posicionado em relação à janela do navegador e não se move quando a página é rolada. 

          
            .div-azul {
              position: fixed;
              right: 0;
              top: 50px;
            }

Position Sticky

Um elemento com position: sticky alterna entre relative e fixed, dependendo da posição de rolagem. Ele é posicionado em relação ao seu contêiner de rolagem mais próximo. Ótimo pra fazer cabeçalhos fixos. 

          
            .div-azul {
              position: sticky;
              top: 0;
            }
