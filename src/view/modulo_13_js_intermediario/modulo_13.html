Windown

Windown é o objeto global do navegador, ele é o objeto que contém todos os outros objetos do navegador, como o document, history, location, navigator, screen, alert, confirm, prompt, etc.

Quando você declara uma variável sem o uso da palavra chave var, let ou const, ela é automaticamente atribuída ao objeto global, ou seja, ao objeto window. 

        
          x = 10;
          console.log(window.x); // 10

document

O objeto document é um objeto que representa o documento HTML que está sendo exibido no navegador. Ele é um objeto filho do objeto window.

O objeto document é o principal objeto do DOM (Document Object Model), ele é o responsável por manipular o conteúdo HTML da página. 

        
          console.log(document); // document

Acessando Elementos

Para acessar um elemento HTML através do JavaScript, você pode utilizar o método getElementById(), que recebe como parâmetro o id do elemento que você deseja acessar. 

        
          const element = document.getElementById('elementId');

O método getElementById() retorna o elemento HTML que possui o id especificado, ou null caso não exista nenhum elemento com o id especificado.

Você também pode acessar um elemento HTML através de sua classe utilizando o getElementsByClassName(). 

        
          const elements = document.getElementsByClassName('className');

Outra forma de acessar um elemento HTML, é através de sua tag html utilizando o getElementsByTagName(). 

        
          const elements = document.getElementsByTagName('tag');

O método getElementsByTagName() retorna uma coleção de elementos HTML que possuem a tag especificada, ou uma coleção vazia caso não exista nenhum elemento com a tag especificada.
