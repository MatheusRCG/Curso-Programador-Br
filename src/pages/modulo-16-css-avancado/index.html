Pseudo Elementos

Pseudo-elementos são palavras-chave adicionadas a seletores que permitem que você estilize partes específicas de um elemento. Eles são usados para aplicar estilos a partes de um elemento que não podem ser acessadas diretamente pelo HTML.

::first-line

O pseudo-elemento ::first-line é usado para aplicar estilos à primeira linha de um bloco de texto. 

          
            p::first-line {
              font-weight: bold;
            }

::first-letter

O pseudo-elemento ::first-letter é usado para aplicar estilos à primeira letra de um bloco de texto. 

          
            p::first-letter {
              font-size: 2em;
              color: green;
            }

Esta é a primeira letra de um parágrafo que está sendo estilizada com ::first-letter.

::after

O pseudo-elemento ::after é usado para inserir conteúdo depois do conteúdo de um elemento. 

          
            li {
              list-style: none;
            }

            li::after {
              content: "'-'";
              color: green;
              font-size: 1.2em;
            }

::before

O pseudo-elemento ::before é usado para inserir conteúdo antes do conteúdo de um elemento. 

          
            li {
              list-style: none;
            }
            
            li::before {
              content: "";
              display: inline-block;
              width: 20px;
              height: 20px;
              background-image: url(./images/google-logo.png);
              background-repeat: no-repeat;
              background-position: center;
              background-size: cover;
            }

Selection

O pseudo-elemento ::selection é usado para estilizar o texto selecionado pelo usuário. 

          
            ::selection {
              background-color: green;
              color: white;
            }
